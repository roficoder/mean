Div and Span

  • Div is a block level element which is provided by html when we don't find any semantic tag
  • Span is an inline element which we use when we want to use inline element

Lists (Ordered vs Unordered)

Unordered Lists (Bullet Lists):

An unordered list is used to create a list of items that do not have a specific order, and each item is marked with a bullet point.

  • ul stands for "unordered list."
  • li stands for "list item."
  • Bullet points (â€¢) are automatically added to the list items.
Ordered Lists (Numbered Lists):

An ordered list is used to create a list of items that have a specific order or sequence, and each item is numbered.

  • ol stands for "ordered list."
  • li stands for "list item."
  • Numeric points (1,2,3) are automatically added to the list items.
