How to Make Web Live

  • Go to your github profile and click on new. It will take you to a create repo page which you can see in second image
  • Here write the name of repo you want in the GREEN BOX. Please make sure to keep repo public, you can see in RED BOX. Click on create repo button in BLUE BOX.
  • You will be landed on this page after creating repo. Upload files by clicking on this button. And please make sure you have index.html in the project otherwise server will not make it live.
  • After uploading files go to setting
  • Go to pages
  • Select main branch
  • Click on save and BOOM
  • After sometime refresh the page and you can visit your site using this link
